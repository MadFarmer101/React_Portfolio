## Getting started
### Dependencies  
* Yarn
* React
* Cypress
* Chance
* Axios


### Setup   
To test this application, fork the repo to your own GitHub account and clone it to your local workspace. </br>

Install all of the dependencies:    
```
$ yarn install
```  
Run the unit tests:  
```
$ yarn test
```  
Start cypress and run the feature tests:  
```
$ yarn run cy:open
$ yarn start
```
